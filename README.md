## Description

NetPractice is a practical networking simulation project from the 42 curriculum that provides hands-on experience with TCP/IP networking fundamentals. 
This web-based interactive tool challenges students to configure network devices to establish functional communication across increasingly complex network topologies.

## Instructions

This project does not need to be compiled. 

A website is provided in the intra, download it and open "index.html" with a web browser.

To export configurations, press "Get my config" button.

Submission requirements:

- 10 JSON files: One config for each level in the website. They represent a solution for a specific level. There are 10 levels in the website
- This markdown file, following strict requirements

## Resources

Here's a list of references consulted for this project:

- CommonMark specification version 0.31.2 (2024-01-28): https://spec.commonmark.org/

- RFC 791 Internet Protocol. The specification for IPv4 addressing, classes, and header structure
- RFC 1518 & RFC 1519 Classless Inter-Domain Routing. Introduction of CIDR notation replacing Classful Network addressing
- RFC 1918 Address Allocation for Private Internets. Defines private IP ranges
- RFC 4632 CIDR Address Strategy. Modern update on CIDR and addressing

- RFC 950 Internet Standard Subnetting Procedure. Original subnetting specification
- RFC 1878 Variable Length Subnet table. Practical guide for subnet calculations

- RFC 1812 Requirements for IP Version 4 Routers. Comprehensive router behavior including routing tables, forwarding, and interfaces
- RFC 792 Internet Control Message Protocol. Explains "ping" functionality
- RFC 826 Address Resolution Protocol. How devices map IP addresses to MAC addresses on LAN

- RFC 793 Transmission Control Protocol
- RFC 768 User Datagram Protocol
- RFC 1180 TCP/IP Tutorial. A beginner-friendly introduction

- RFC 1122 Requirements for Internet Hosts. Defines how hosts should behave, including default gateway configuration

- My notes in Notion

AI was not used for any task or any part of the project.
